# Basic Django Project

This is a simple Django web application that includes:

- Views (`views.py`)
- Templates (`templates/`)
- URLs (`urls.py`)
- Static files (`static/`)

---

## 🚀 Features

- Home page rendered through a template
- Basic URL routing
- Static CSS file linked to the template
- Organized project structure
